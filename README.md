# Mint-ui
Vue2.X + Mint-ui
####vue-cli搭建项目
* 全局安装vue-cli
<br>
`cnpm install vue-cli ` 
* 创建一个基于webpack的新项目
<br>
`vue init webpack projectName`
* 安装依赖
<br>
`cd projectName`
`cnpm install`
* 启动项目
<br>
`cnpm run dev`

####Mint-UI 配置
* 安装插件
<br>
`cnpm install mint-ui -S`
* 在main.js文件中全局引入
<br>
`import MintUI from 'mint-ui`
<br>
`import 'mint-ui/lib/style.css'`
<br>
`Vue.use(MintUI)`
